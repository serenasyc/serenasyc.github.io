The Four-Card Problem

The Four-Card Problem, more commonly known as the Wason selection task, is a logic puzzle created by Peter Cathcart Wason in 1966. The problem is stated below.

You are shown the four cards below, each with a number on one side and a colour on the opposite side. I tell you that the cards follow this rule, "If a card shows an even number, the opposide face is red." Which card or cards do you need to flip to determine if this rule is true?

Click the cards for a hint

3
8

In classical logic, we express the rule as an implication. The truth table of an implication is shown below

p = "One face is even". q = "The opposite face is red." If p, then q.
True True True
True False False
False True True
False False True

The rule is only false in the case where one face of the card is even while the opposite face is not red.

We flip over the 8 card because 8 is an even number.
If the rule is true, the opposite face of the 8 card is red.

We also flip over the brown card.
If the rule is true, the opposite face of the brown card cannot be even.
